Step-by-Step Calibration Procedures

Bed Leveling

  1. Home the printer using G28
  2. Use manual adjusters or BLTouch to level the bed
  3. Verify with a piece of paper between nozzle and bed

Z-Offset Adjustment

  1. Print a test cube at 0.3mm layer height
  2. Measure cube height with calipers
  3. Adjust Z-axis using M295 command if discrepancy exists

Extruder Calibration

  1. Extrude 100mm filament using G1 E100 F50
  2. Measure actual extruded length
  3. Adjust steps/mm in firmware if discrepancy >2%
