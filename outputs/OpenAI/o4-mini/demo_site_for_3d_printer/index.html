Overview of 3D Printing Technologies

3D printing, or additive manufacturing, builds parts layer by layer from digital models. Common desktop technologies include:

  • FDM/FFF (Fused Deposition Modeling/Fused Filament Fabrication): Melts and extrudes filament.
  • SLA (Stereolithography): Cures resin with UV laser or projector.
  • SLS (Selective Laser Sintering): Fuses powdered materials with a laser.

This guide focuses on FDM/FFF processes and best practices.

Calibration Procedures

1. Bed Leveling

  1. Preheat bed and nozzle to printing temperatures (e.g., 606C bed, 2006C nozzle).
  2. Disable steppers and move the nozzle to each corner and center.
  3. Insert a piece of paper between nozzle and bed; adjust leveling screws until slight friction is felt.
  4. Repeat the process 2–3 times to ensure consistency.

2. Extruder Steps/mm Calibration

Ensure accurate filament extrusion:

1. Mark 100 mm on the filament above the extruder entry.
2. Extrude 100 mm of filament via printer controls.
3. Measure actual extruded length.
4. Calculate new steps/mm:
   new_steps = (current_steps * 100) / actual_extruded_mm
5. Update in firmware or via M92 command:
   M92 E[new_steps]
   M500

3. PID Tuning

M303 E0 S200 C8   ; Run autotune for hotend at 200°C for 8 cycles
M500              ; Save results to EEPROM

Replace E0 with E-1 for the bed.

Filament Change Process

  1. Preheat nozzle to target temperature (e.g., 2006C for PLA).
  2. Retract existing filament: Use 100% feedrate, retract ~100 mm.
  3. Remove old filament spool.
  4. Cut the new filament at a 45° angle to improve feed.
  5. Insert new filament until you see molten plastic extruding.
  6. Extrude 10–20 mm to purge residual old material.

Post-Filament Change Checklist

  • Verify extrusion: Ensure smooth, continuous flow without gaps.
  • Check first layer adhesion: Print a test line or skirt.
  • Adjust Z-offset if needed: Move the nozzle closer or farther from the bed.
  • Run a small calibration cube: 20×20×20 mm test print.
  • Inspect for under- or over-extrusion: Tune flow rate or steps/mm.

Troubleshooting Common Issues

Poor First Layer Adhesion

  • Re-level the bed.
  • Clean bed surface with isopropyl alcohol.
  • Use adhesive aids: Glue stick, tape, PEI sheet.

Stringing or Oozing

  • Enable retraction: 5–7 mm at 40–60 mm/s.
  • Lower nozzle temperature by 5–106C.
  • Enable coasting in slicer.

Layer Shifting

  • Tighten belt tensions.
  • Check pulleys and set screws on stepper shafts.
  • Reduce print speed if necessary.
