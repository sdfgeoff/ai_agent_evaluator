3D Printer Technical Guide

Introduction

This guide covers essential maintenance procedures for 3D printers, including calibration and post-filament change processes.

Calibration Procedures

  1. Bed Leveling: Use a piece of paper to adjust bed height at each corner until even resistance is felt.
  2. Nozzle Alignment: Run a test print with alignment cubes to check X/Y axis accuracy.
  3. Extruder Calibration (E-steps): Measure filament output for 100 mm movement and adjust steps/mm in firmware if needed.
  4. Temperature Tuning: Test print with different temperatures (190-230°C for PLA) to achieve optimal flow.

Post-Filament Change Process

  1. Purge 10-20 cm of new filament to remove residue from old material.
  2. Check nozzle temperature matches new filament requirements (e.g., PLA: 190°C, ABS: 230°C).
  3. Perform a test extrusion (50 mm) to ensure smooth flow.
  4. Inspect for stringing - if present, increase retraction distance or speed in slicer settings.

Conclusion

Maintaining proper calibration and following post-filament procedures ensures print quality and prevents common issues like under-extrusion or bed adhesion failure.
